# نظام إدارة الطلاب — Student Management System

## تثبيت وتشغيل

```bash
pip install flask flask-sqlalchemy
python app.py
```
ثم افتح: http://localhost:5000

## بيانات الدخول التجريبية
| المستخدم | كلمة المرور | الصلاحية |
|----------|-------------|---------|
| admin | admin123 | مدير النظام |
| teacher | teacher123 | معلم |

## الصفحات
- `/` — تسجيل الدخول
- `/dashboard` — لوحة التحكم + إحصائيات
- `/students` — قائمة الطلاب
- `/students/add` — تسجيل طالب جديد
- `/attendance` — تسجيل الحضور والغياب
- `/reports` — التقارير + إرسال واتساب
